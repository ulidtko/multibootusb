MultiBoot USB

This is a collection of GRUB scripts that allows to create a bootable pendrive (USB dongle, flash stick) supporting a number of different OS images. Simply copy ISO files under `/iso-imgs`. Both UEFI and Legacy BIOS boot modes can be enabled simultaneously.

RFRemix

Russian Fedora Remix is a remix of the Fedora Linux distribution adapted for Russia. However, it is neither a copy of the original Fedora nor a new GNU/Linux distribution. The project aims to ensure that Fedora fully satisfies the needs of Russian users with many additional features provided out of the box (e.g., specific software packages, pre-installed drivers for popular graphics processors, manuals in Russian).
