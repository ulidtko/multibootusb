MultiBoot USB

This is a collection of GRUB scripts that allows to create a bootable pendrive (USB dongle, flash stick) supporting a number of different OS images. Simply copy ISO files under `/iso-imgs`. Both UEFI and Legacy BIOS boot modes can be enabled simultaneously.

SpinRite

SpinRite is a computer program for scanning magnetic data storage devices such as hard disks, recovering data from them and refreshing their surfaces. The first version was released in 1987 by Steve Gibson. Version 6.0, still current as of June 2017, was released in 2004. SpinRite is run from a bootable medium (such as a CD, DVD or USB memory stick) on a PC-compatible computer, allowing it to scan a computer’s hard drive and file system.
